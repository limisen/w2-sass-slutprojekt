Medlemsförmåner

När man tänker på allt man får genom att vara medlem i ABC-klubben är det egentligen konstigt att det inte kostar mer.

Rabatt på datortidskrifter

Medlemskapet ger 25% rabatt på tidskriften PC för alla. Beställ prenumeration enklast via denna länk.

Personlig e-postadress

Som standard får du två adresser. Dels medlemsnummer@abc.se och dels fornamn.efternamn@abc.se. Möjligheter finns givetvis också att beställa egna alias till adressen.

Din post hämtar du via POP- eller IMAP med vanliga e-postprogram, via vår webmail eller direkt på klubbens servrar med Pine eller något annat Unix-program.

För att sända e-post använder du SMTP AUTH och sänder via vår server. Instruktioner för detta finns under medlemsinfo.

Mailserver utan spionage

Vi läser eller analyserar aldrig våra medlemmas mejl. De flesta stora mejlservrar, t.ex. google gmail, analyserar dina mejl för att kunna ge dig riktade annonser, och för att upptäcka spam. Det slipper du hos oss. Vi har spamskydd enligt de principer som beskrivs här, dvs. grålistning och spamtrapping, och det innebär att vi aldrig analyserar innehållet i mejlen.

Egen webbplats på våra servrar

Om du vill kan du lägga upp en egen webbplats på någon av våra servrar. Vi kör Apache (OpenBSD). För dig som behöver finns stöd för bl.a. PHP och CGI.

Parkera egen domän hos klubben

För dig som har eller skaffar en egen domän finns möjligheten att parkera den hos oss. Det innebär att du har din domän någonstans i vårt system och att den är registrerad i våra DNS-servrar.

Skalkonto i klubbens UNIX-system

Alla medlemmar får inloggnings­uppgifter till vårt UNIX-system och ett eget hembibliotek att lägga sin filer Vi har inte lagt några begränsningar på hur mycket utrymme du får belägga (det finns så det räcker), men vi vill att du säger till om du behöver mer än några GB.

Unixkontot får du som standard.

Webbproxy

Vi har en webbproxy (squif) på orm som våra medlemmar kan använda. Den lyssnar på port 3128 på localhost på orm. För att använda den behöver du sätta upp en ssh-tunnel från din dator till orm.abc.se:

$ ssh -L 3128:127.0.0.1:3128 orm.abc.se

Alternativt stoppa in följande i din ~/.ssh/config på hemmadatorn:

Host orm.abc.se
LocalForward 3128 127.0.0.1:3128

... och sedan starta chrome såhär: chrome --proxy-server=127.0.0.1:3128 alternativt: chrome --incognito --proxy-server=127.0.0.1:3128

Verifiera att det fungerar genom att surfa till https://www.abc.se/~lln/ip/ Den IP-address som står där bör vara 158.174.61.226.

Vi har ingen loggning alls vilka som använder den eller till vart någon surfar. Det är "access_log none"-direktivet i /etc/squid/squid.conf på orm. Var vänlig och missbruka inte detta.

Vi har planer på att implementera adblocker för din webbrowser via DNS.
